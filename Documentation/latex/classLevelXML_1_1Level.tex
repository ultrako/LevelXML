\doxysubsection{Detailed Description}
The \doxylink{classLevelXML_1_1Level}{Level} class represents a Happy Wheels level. 

As this extends \doxylink{classLevelXML_1_1LevelXMLTag}{Level\+XMLTag}, \doxylink{classLevelXML_1_1Level_af04d84d85a45ced9153529b0f2230c67}{To\+XML()} will print out the \doxylink{namespaceLevelXML}{Level\+XML} that represents this level, ready to be pasted into the Happy Wheels import box.
