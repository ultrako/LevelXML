Detailed Description

This class represents any xml tag that can go into the import box in Happy Wheels.

If it makes sense for a class to be made outside of a level, it will have a constructor that takes a string. All public classes that inherit this will have a public empty constructor, and you will end up with an object that would be the same as if you had opened the level editor and placed that object without changing anything
